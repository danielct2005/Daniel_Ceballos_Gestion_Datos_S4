# SalmonttApp - Gestión de Centros de Cultivo

Proyecto de la Semana 4 para leer datos de producción de salmón desde un archivo de texto.

## Funcionalidades
1. Carga de datos desde `resources/centros.txt`.
2. Creación dinámica de objetos `CentroCultivo`.
3. Listado completo de centros.
4. Filtrado de centros con producción superior a 1000 toneladas.
